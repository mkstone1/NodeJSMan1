2. Javascript basics

Preface

This section assumes some familiarty with javascript. As such objects, arrays, callback functions etc are not covered here.

2.1 Variables

There are 3 ways to declare a variable in javascript. These are const, let and var

2.1.1 Var

Variables declared using var are either globally scoped or functionally scoped. If we look at the code below we can see that we can have a var variable defined in the global scope and in the function of getColor. However var does not support block level scoping. This means that when we redeclare color in the if statement it changes the one declared in the global scope. It is also possible to reassign the variable later on.

As a general rule you should not use var

2.1.2 Let

The let keyword can function very similar to the var keyword. The can both be designed later on. The main difference is that let supports block level scoping and cannot be redeclared.

2.1.3 Const

Const and let share the block level scoping however cannot redeclare or reassign const variables. Therefore they must be assigned a value at declaration. You should use the const keywork as much as possible and let when needed

2.2 Loops

Normal for and while loops should no longer be used unless used for "hand counting". You should use the functional approch and use the loop functions decriped here

2.2.1 Loop function

Here are the loop function that should be used. They all accept a callback funtions

  • Map - Return an array of the same length. You can change the original data, mapping the orginal data to something else
  • Filter - Returns an array with less items than the original array
  • Reduce - Returns a single value which can also be an object
  • Find - Returns a single items. This will the first item that satifies some condition
