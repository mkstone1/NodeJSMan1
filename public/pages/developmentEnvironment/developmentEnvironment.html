1. Setting up your development environment

1.1 Donwload NodeJS

The first thing needed is to go and download Node JS. The easiest way is to click this link and download the installer for your OS

1.2 Download and IDE

Go and download your favorite IDE for javascript. I can recommend visual studio code which can be downloaded here

1.3 Setup folder for your project

You will need to store all files for your project somewhere on your PC. Find some place and create a folder

1.4 Initializing project

When setting up your project there is 2 ways to go about it. You can either manually set it up or do it with npm commands

Manually Initializing your project

In your project folder create a file called package.json. Here you will list all the dependencies for your project. Here we use express as our web application framework and nodemon for auto refresh when updating code files. Your package.json should look like this

{"dependencies":
{ "express": "^4.18.2",
"nodemon": "^2.0.21"}
}

With your dependcies setup you can now run npm i from a terminal to download dependencies

Automaticlly setting up your development environment

Open a terminal and go to your project folder. Run npm init. Follow the prompts. Most should be self explanatory. For entry point write: app.js Afterwards do npm i express and npm i nodemon
Lastly create an app.js in the root of your project


You are now ready to start coding with Node JS and express. To start your project write nodemon.cmd app.js in the terminal
